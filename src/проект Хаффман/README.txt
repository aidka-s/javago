Алгоритм Хаффмана

1. Компиляция и запуск
bash
javac TestRunner.java
java TestRunner              # запуск всех тестов
java TestRunner encode input.txt encoded.bin
java TestRunner decode encoded.bin output.txt

2. Три тестовых примера
test1.txt - 10 одинаковых символов "1"

test2.txt - 20 символов: 10x'1', 5x'2', 5x'3'

TestRunner.class - бинарный файл программы

3. Формат закодированного файла
text
[4 байта] - размер таблицы частот
Для каждого символа:
  [1 байт] - символ
  [4 байта] - частота
[4 байта] - количество битов данных
[n байт] - закодированные данные

4. Пример вывода тестов
text

Тест 1: 10 символов '1'
Исходный: 10 байт, Закодированный: 15 байт, Сжатие: 1.50
✓ Корректно

Тест 2: 20 символов
Исходный: 20 байт, Закодированный: 29 байт, Сжатие: 1.45
✓ Корректно

Тест 3: Бинарный файл
Исходный: X байт, Закодированный: Y байт, Сжатие: Z
✓ Корректно
